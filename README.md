# Gryffin Tech Website

A modern, responsive business website built for **Gryffin Tech**, focused on presenting the company’s digital services, brand identity, and online presence in a clean and professional way.

🌐 **Live Demo:** https://gryffintech.netlify.app/

---

## 📌 Overview

This project is a front-end web application developed using **React.js**. It serves as the official website for Gryffin Tech and is designed to provide visitors with a smooth, informative, and visually engaging browsing experience.

The website includes a professional landing page structure, reusable React components, styled sections, and a responsive layout suitable for desktop and mobile users.

## 🚀 Getting Started

Follow these steps to run the project locally.

### 1. Clone the repository

```bash
git clone https://github.com/HashirBaig/Gryffintech-Web.git
```

### 2. Navigate into the project folder

```bash
cd Gryffintech-Web
```

### 3. Install dependencies

Using Yarn:

```bash
yarn install
```

Or using npm:

```bash
npm install
```

### 4. Start the development server

```bash
yarn start
```

Or:

```bash
npm start
```

The application will run locally at:

```bash
http://localhost:3000
```
